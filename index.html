¡Calcula el factorial de un número!

El factorial de un número es una operación matemática que se representa con el símbolo !. El factorial de un número entero positivo n se define como el producto de todos los enteros positivos desde 1 hasta n. Es decir:

n!=n x (n-1) x (n-2) x ⋯ ⋯ x1
